# Run for Android

Ever wish you could just enter a web address or file path on your phone, just like pressing `Win+R` on a Windows PC, and instantly jump to it?

**Run** is exactly that — a minimalist quick-launch tool for Android.

---

## 💡 Inspiration

No complex story. Just a very simple idea: **bring the classic, clean "Run" window from a Windows PC onto an Android phone.** That's it.

---

## ⚠️ Current Status (Early Development)

This project is in **very early development**. Right now, it's just a bare-bones input window. It doesn't look polished yet, and features are being built step by step.

If you try it now, please treat it as an early test build.

---

## ✅ What It Can Do Right Now

Just enter something into the input box and tap **"OK"**:

*   **Enter a web address (e.g., `www.bilibili.com`)**: It will automatically open the page in your browser.
*   **Enter a file path (e.g., `/sdcard/Download/notes.txt`)**: It will automatically open the file with the corresponding app on your phone.

> 💡 **Note on Apps**: Currently, you can also enter an **"app package name"** (like `com.google.android.run`) to launch an app.
> This is **only a temporary workaround** for the early stage. The visual "App Picker" window hasn't been built yet, so you have to rely on package names. In future releases, you'll be able to select apps from a simple visual list — no more memorizing those complicated English strings.

---

## 🗺️ Future Plans

We plan to gradually polish this "bare shell" into a complete tool:

*   🖱️ **App Picker Window**: Select apps directly from a list instead of typing package names.
*   📁 **File Browser Button**: Browse and pick files directly from your phone's folders.
*   🎨 **UI Overhaul & Dark Mode**: Better visual design and automatic dark theme support.
*   ⚠️ **Security Alerts**: Pop-up warnings when attempting to open certain potentially risky apps.
